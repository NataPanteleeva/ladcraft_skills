Шаблон показывает корректную связку:

- native handler tool
- `return data` + widget binding
- EJS widget
- `resources.network.hosts`

Что можно копировать как есть:

- паттерн возврата данных для widget

- данные и тексты
- список внешних хостов

- не использовать Handlebars blocks
- не добавлять TypeScript/JSDoc-типы в `scripts/*.js`
- не забывать добавить все внешние хосты в `resources.network.hosts`
