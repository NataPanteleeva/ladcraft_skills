# Reference only: `migrated-skill`

Этот каталог показывает, как выглядит уже собранный payload после publish-конверсии.

Используйте его только для сверки:

- структуры `tools[]`
- итогового `function`
- `environment`, `resources`, `schemas`

Не используйте этот каталог как стартовый шаблон для новых навыков и не переносите оттуда TypeScript/JSDoc-подходы в `scripts/*.js`. Для новых навыков берите approved templates: `minimal-skill`, `vfs-skill-example`, `widget-skill-example`, `environment-user-example`.
