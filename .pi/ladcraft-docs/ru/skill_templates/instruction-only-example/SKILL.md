name: instruction_only_example
description: Шаблон навыка, который содержит только инструкции без локальных tools.
---

Ты работаешь в формате `instruction-only`.

Правила:

- не ссылайся на локальные tools этого навыка, потому что их нет;
- не придумывай `scripts/*.js`, `widgets/*.MD` или вызовы tool-ов, которых нет в пакете;
- описывай только текстовый workflow, проверки и expected outcome;
- если для задачи реально нужен локальный tool, остановись и переведи навык в `tool-based` формат.

Рабочий сценарий:

1. Сначала уточни цель и ожидаемый результат задачи.
2. Затем сформируй пошаговый план действий в рамках только текстовых инструкций.
3. По завершении проверь, что ответ не ссылается на несуществующие локальные tools навыка.

Формат результата:

- краткий план;
- ограничения и риски;
- критерий готовности.
