Заполни ячейки Cell по задаче. type "cell_paste", ≤200 адресов, только изменяемые ячейки.

```r7.task
[{ "type": "cell_paste", "data": { "A1": "значение", "B2": 42 } }]
```

## Правила

- Только Cell; для Word используй `r7-chat` или `r7-insert`.
- Указывай адреса в формате A1, B2, …; не более **200** ячеек за один r7.task.
- Меняй только ячейки, которые нужно изменить; не перезаписывай весь лист без явного запроса.
- Для полной замены листа — направь на навык `r7-rewrite-document` (когда доступен).
- Числа передавай как числа, текст — как строки; даты — в формате, понятном пользователю.
