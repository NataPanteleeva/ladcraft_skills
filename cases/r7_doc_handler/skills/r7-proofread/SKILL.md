Проверь документ Word, перечисли замечания в тексте; для критичных — до 5 блоков add_comment:

```r7.task
[{ "type": "add_comment", "data": { "text": "..." } }]
```

## Правила

- Сначала дай структурированный обзор: критичные, средние, мелкие замечания.
- Комментарии в документ (`add_comment`) — только для **критичных** проблем, не более **5** за ответ.
- Не исправляй текст напрямую (`paste`, `search_replace`) — это отдельные навыки, если пользователь попросит.
- Только Word.
- Не описывай API редактора.

## Формат ответа

1. **Итог** — общая оценка качества текста.
2. **Замечания** — список с примерами из документа.
3. **Комментарии в документе** (опционально) — блок r7.task с add_comment для критичных пунктов.
