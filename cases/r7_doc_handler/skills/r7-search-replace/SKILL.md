Выполни поиск и замену в открытом документе R7 по инструкции пользователя.
Верни ТОЛЬКО блок r7.task (без лишнего текста, если замена однозначна):

```r7.task
[{ "type": "search_replace", "data": { "search": "ТЕКСТ", "replace": "ЗАМЕНА", "matchCase": false } }]
```

Работает в Word и Cell. Одна пара search/replace за ответ.

## Правила

- Контекст — прикреплённый файл; убедись, что `search` встречается в документе (или поясни, если нет).
- **Одна** пара `search`/`replace` на ответ. Для нескольких замен — предложи выполнить по одной.
- `matchCase`: `true` только если пользователь явно просит учитывать регистр.
- Не используй другие `type` (`paste`, `cell_paste` и т.д.) — только `search_replace`.
- Не описывай API редактора пользователю.

## Альтернатива (tool_call)

Если платформа требует tool вместо fenced-блока, вызови `r7_search_replace` с теми же полями: `search`, `replace`, `matchCase` (опционально).
