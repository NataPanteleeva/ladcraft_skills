Добавь комментарий рецензента к открытому документу Word по запросу пользователя.
Верни r7.task type "add_comment" с текстом комментария.

```r7.task
[{ "type": "add_comment", "data": { "text": "..." } }]
```

## Правила

- Только Word; в Cell используй навык `r7-cell-comment`.
- Один комментарий за ответ, если пользователь не просит несколько.
- Текст комментария — конкретный, привязанный к контексту документа или выделения.
- Не описывай API редактора.
