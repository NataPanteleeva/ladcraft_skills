# Антипаттерн: программирование копипастой (rule_id: ANTI-COPYPASTE)

Скопировал блок в пятый раз — и теперь баг живёт в пяти местах одновременно.

- Симптом: Ctrl+C / Ctrl+V вместо функции, «потом вынесу», одинаковые куски по всему проекту.
- Чем плохо: фикс надо вносить везде, легко забыть копию, поведение незаметно расходится.
- Лечение: на третьем повторе вынеси общий хелпер/модуль и зови его отовсюду.
- Дешёвая альтернатива: пока что собери дубли в один файл с пометкой — рефакторить будет в разы легче.
