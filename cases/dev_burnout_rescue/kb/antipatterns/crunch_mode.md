# Антипаттерн: вечный кранч (rule_id: ANTI-CRUNCH)

«Ещё один спринт на износ, а потом отдохнём». Этот «потом» почему-то не наступает.

- Симптом: переработки как норма, выходные в офисе, «отдых после релиза» переносится бесконечно.
- Чем плохо: команда выгорает, скорость падает, лучшие тихо уходят, качество проседает.
- Лечение: фиксируй устойчивый темп, режь скоуп честно, защищай выходные команды.
- Дешёвая альтернатива: один разгрузочный день в спринте окупается меньшим числом багов.
