# Совет: перегруз встречами (rule_id: ADV-MEETINGS)

**Суть.** Если день — это встречи, код пишется ночью, и тогда выгорание гарантировано. Время на фокус надо защищать явно.

**Шаги:**
1. Половину встреч можно заменить асинхронным тредом — предложи это для статусов.
2. Заблокируй в календаре 2 «фокус-блока» в день как реальные встречи.
3. Перед встречей спрашивай повестку; нет повестки — нет смысла.
4. Группируй встречи в один отрезок дня, оставляя длинные непрерывные слоты.

**Варианты по ситуации:**
- Встреча без повестки и решений — вежливо откажись или попроси краткое резюме в тексте.
- Ты не принимаешь решений на встрече — попроси протокол вместо личного присутствия.
- Встреч объективно много из-за роли — обсуди с менеджером «день без встреч» для всей команды.

**Когда эскалировать:** календарь забит не тобой и не двигается — это разговор с менеджером о приоритетах.
